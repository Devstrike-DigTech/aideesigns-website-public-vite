# Terms of Service Content Example

Create a new page in Admin Dashboard → CMS with these details:

## Page Settings
- **Slug:** `terms`
- **Title:** `Terms of Service`

## Content Blocks to Add

### Block 1: Introduction (Text Block)
**Block Key:** `intro`
**Block Type:** `text`
**Content:**
```

Please read these Terms carefully before using our services. If you do not agree with these Terms, you may not use our services.
```

---

## How to Add in Admin Dashboard

1. Go to **Admin Dashboard** → **CMS**
2. Click **"Create Page"**
3. Enter:
   - Slug: `terms`
   - Title: `Terms of Service`
4. Click **"Create Page"**
5. Click **"Edit Content"** on the new page
6. Add each block listed above
7. Visit **http://localhost:3000/terms** to see your page!
